# wilddrone-hackathon-2025-bvlos-ops
Official documentation archive for the WildDrone BVLOS Hackathon conducted at Ol Pejeta Conservancy (Kenya) in January 2025.  The repository includes all permits, safety procedures, and supporting documents used to plan, execute, and assess Beyond Visual Line of Sight (BVLOS) UAS operations under the SORA 2.5 framework.
